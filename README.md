# MKVMerge GUI v2

PySide6-версия утилиты для пакетной обработки MKV-файлов (замена аудиодорожек через mkvmerge).

## Требования

- **Python 3.10+**
- **PySide6** — графическая библиотека (Qt для Python)
- **mkvmerge** — должен быть установлен и путь прописан в настройках
- **WinRAR** или **7-Zip** — для распаковки RAR архивов (опционально)

## Установка

### Шаг 1. Установить Python

**Способ 1** — через winget (в PowerShell от администратора):
```
winget install Python.Python.3.13
```

Если появилась ошибка `winget : Имя "winget" не распознано как имя командлета` — значит winget не установлен. Используйте **Способ 2**.

**Способ 2** — скачать установщик с https://www.python.org/downloads/ (кнопка **Download Python 3.x.x**). При установке **обязательно** поставить галочку **"Add Python to PATH"** (внизу окна установщика).

> **Важно:** если PowerShell был открыт до установки Python — закройте его и откройте заново (от администратора). Уже открытый PowerShell не увидит новые команды `python` и `pip`.

### Шаг 2. Установить зависимости

В PowerShell (от администратора):
```
pip install PySide6
```
Это графическая библиотека (Qt для Python) — **обязательна** для работы программы.

### Шаг 3. Определение длительности видео (опционально)

Если хотите видеть длительность видеофайлов в программе:

1. Скачать и установить **MediaInfo** с https://mediaarea.net/en/MediaInfo/Download (выбрать **Windows** → **EXE**).

2. После установки MediaInfo выполнить в PowerShell (от администратора):
   ```
   pip install pymediainfo
   ```

Без этого шага программа будет работать, но не будет показывать длительность видео.

### Шаг 4. Распаковка архивов (опционально)

Для распаковки RAR-архивов нужен **WinRAR** или **7-Zip**:

- WinRAR: https://www.win-rar.com/download.html
- 7-Zip: https://www.7-zip.org/download.html

Программа автоматически ищет `UnRAR.exe` в `C:\Program Files\WinRAR\` и `7z.exe` в `C:\Program Files\7-Zip\`.

## Запуск

Положить `mkvmerge_gui_v2.pyw` в **отдельную папку**. При запуске рядом с ним автоматически создадутся папки `config_films/` и `config_settings/` с настройками. Все изменения сохраняются на диск автоматически каждую секунду.

Двойной клик по файлу `mkvmerge_gui_v2.pyw`.

Если двойной клик не запускает программу — выполнить в PowerShell:
```
python mkvmerge_gui_v2.pyw
```

### Бэкапы

В папке `config_films/` автоматически создаются резервные копии базы фильмов — это нормально, не удаляйте их:

- **films.json.bak1 — .bak5** — последние 5 копий, обновляются при каждом сохранении
- **films.json.daily_\*** — дневные бэкапы раз в 12 часов (хранятся 10 штук)
- **films.json.safe_\*** — аварийные бэкапы, создаются автоматически если программа обнаружила потерю данных
